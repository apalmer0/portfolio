Featured Project

Bring your images to life

flipbook allows a user to upload images and draw sketches to create frames that can be compiled into an animated gif, like a flipbook. If no frames are selected, the user can use their webcam to create the GIF of themselves.

For this project I wanted to experiement with some technologies I wasn't really comfortable with yet; despite having just finished upBucket I was still a shaky on Express and Node, so those were the first choices for the API. On the front end, I've been curious about Angular for a long time, so I was excited to teach it to myself in order to finish this project.
