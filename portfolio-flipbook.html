flipbook

Bring your images to life

flipbook allows a user to upload images and draw sketches to create frames that can be compiled into an animated gif, like a flipbook. If no frames are selected, the user can use their webcam to create the GIF of themselves.

For this project I wanted to experiment with some technologies I wasn't really comfortable with yet; despite having just finished upBucket I was still a little shaky on Express and Node, so those were the first choices for the API. On the front end, I've been curious about Angular for a long time, so I was excited to teach it to myself so I could use it for this project.

  • Date: March 30 - April 8 2016
  • Technologies: Angular, Amazon Web Services, Express, MongoDB, Node
